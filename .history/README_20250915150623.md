## 🚀 快速开始

### 安装脚本

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击对应脚本的链接直接安装
3. 或者复制脚本代码手动安装

## 🛠️ 开发指南

### 脚本开发规范

1. 每个脚本必须包含完整的元数据
2. 代码需要有适当的注释
3. 必须包含对应的README文档
4. 遵循统一的命名规范

### 贡献指南

1. Fork 本项目
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
